# 当锈蚀遇见代码：用 Claude 构建赛博废土的诗意

> 一个关于美学、AI 和虚拟团队的实验

## 序章：废墟中的霓虹

2019 年，《银翼杀手 2049》上映。

Roger Deakins 用镜头告诉我们：未来不是光鲜亮丽的，而是破碎的、锈蚀的、被遗弃的。但在这片废墟中，霓虹灯依然闪烁，全息投影依然浮现，人类依然生存。

**这就是赛博废土美学的核心：在衰败中寻找美，在破碎中看到力量。**

三年后，我开始思考一个问题：

**如果 AI 也能理解这种美学，如果代码也能创作这种诗意，会是什么样子？**

这篇文章，记录了我用 Claude Code 构建一个"虚拟电影团队"的过程——不是为了生成视频，而是为了**让美学可以被编码，让诗意可以被传递**。

---

## 第一幕：美学的基因密码

### 破碎的和谐：工业废土机甲美学

赛博废土美学建立在三组矛盾之上：

**破碎 ↔ 增强**

- 锈蚀的金属，镶嵌着发光的电路
- 破损的外壳，内部是精密的液压系统
- 废弃的工厂，墙上投射着全息广告

**有机 ↔ 机械**

- 血肉之躯，接驳着金属义肢
- 布料的柔软，与钢铁的冰冷
- 泥土和油污，覆盖在齿轮和活塞上

### 色彩：冷暖甚至无需对话，而是对峙

我们从经典的工业废土风格中提取了两组核心配色，这不仅是颜色的堆砌，更是情感的表达：

> [!NOTE]
> **主色调：工业黄 (#FFD700) vs 海军蓝 (#0066CC)**
>
> - **工业黄**：代表警示、能量、甚至是一丝危险的活力。它是废土上醒目的伤疤。
> - **海军蓝**：代表技术、深沉、冰冷的秩序。它是覆盖在伤疤上的冷酷绷带。

> [!TIP]
> **辅助色：霓虹橙与赤红**
>
> - **霓虹橙 (#FF8C00)**：温暖、硬核，如同余烬。
> - **赤红 (#FF0000)**：危险、暴力、或者是心脏跳动的颜色。

### 构图：巨物与微尘

在构建画面时，我们遵循三大构图原则：

1. **垂直构图（Scale）**：利用巨大的机甲与渺小的人类形成强烈对比，突显压迫感与敬畏感。
2. **前景主体（Foreground）**：废墟、碎石或破损的零件作为前景，增加画面的层次感和窥视感。
3. **背景衬托（Background）**：远处模糊的霓虹城市或巨大的工厂轮廓，暗示着更广阔的世界观。

这些美学原则，不再只是脑海中的画面，它们将被编入代码。

## 第二幕：当美学遇见 AI

### 问题：美学很美，但 AI 不懂

当我开始用 AI 生成视频时，我发现了一个问题：

**AI 不理解"赛博废土美学"。**

你告诉它"cyberpunk, mecha, neon lights"，它会给你一个光鲜亮丽的机器人。

但你想要的是：

- 锈蚀的装甲板
- 暴露的液压管
- 油污和划痕
- 不对称的修补痕迹

**AI 需要的不是概念，而是具象化的描述。**

### 解决方案：编码美学

我开始思考：**如果把美学原则转化成代码，会怎样？**

不是写一个"提示词生成器"，而是**构建一个理解美学的系统**。

这就是 Claude Code Skill 的起点。

## 第三幕：构建虚拟电影团队

### 灵感：真实的电影制作

真实的电影制作是团队协作：

- 导演负责整体叙事
- 摄影指导负责画面
- 美术指导负责视觉风格
- 灯光师负责光影
- 调色师负责后期

**那么，为什么不让 AI 也这样工作？**

### 团队架构：六个专业角色

我用 Claude Code 构建了六个"虚拟角色"，每个角色都是一个独立的 Skill 模块：

#### 1. 故事架构师（Story Architect）

**美学理念：** 好的视觉需要好的故事支撑。
**职责：** 提供故事框架，如《最后的补给线》或《地下觉醒》。

#### 2. 分镜导演（Storyboard Director）

**美学理念：** 单个镜头可能很美，但组合起来需要叙事性。
**职责：** 将故事拆解为 7 个标准镜头：建立、登场、触发、行动、高潮、解决、余韵。

#### 3. 视觉诗人（Visual Poet）

**美学理念：** AI 需要具象化的描述，而不是抽象概念。
**职责：** 描绘画面细节融合我们定义的“工业黄”与“海军蓝”配色。
**Prompt 示例：**
> "锈橙色的金属表面，油漆剥落露出多层涂装，#FFD700 的警告条纹在 #0066CC 的冷光下显得格外刺眼..."

#### 4. 摄影指导（Cinematographer）

**美学理念：** 运镜决定了观众的视角，转场决定了情绪的流动。
**职责：** 设计专业运镜（Dolly Push-In）和叙事性转场（Match Cut）。

#### 5. 大师顾问（Master Consultant）

**美学理念：** 质量控制，确保每个镜头达到电影级标准。
**职责：** 引入 Roger Deakins 的光影哲学和 Denis Villeneuve 的巨物构图。

#### 6. 模型优化师（Model Optimizer）

**美学理念：** 不同的 AI 模型有不同的"语言"。
**职责：** 适配 SeaDance 2.0（即梦）和 Kling 3.0（可灵）的提示词格式。

## 第四幕：小白也能懂的 Skill 使用指南

### 什么是 Skill？

想象一下，Claude 本来是一个博学多才但有点“泛而不精”的实习生。
**Skill 就像是一张特定的“能力卡片”或者“武功秘籍”。**
当你把这张卡片插进 Claude 的“大脑”里，它瞬间就变成了那个领域的专家——在这个案例里，它变成了一个好莱坞级别的视觉导演团队。

### 如何给你的 Claude 安装“赛博导演”能力卡？

不需要编程基础，只需要三步：

1. **获取秘籍（下载文件）**
    下载我们准备好的 `mecha-story-generator` 文件夹（或者只是里面的 `SKILL.md` 文件）。

2. **装备能力（放入文件夹）**
    找到你电脑上的 Claude Skill 文件夹。
    - **Mac 用户**：通常在 `~/.claude/skills/`
    - **如果没有**：就在你常用的工作目录下新建一个文件夹叫 `skills`。

3. **念出咒语（开始使用）**
    打开你的 Claude Code 终端，输入以下命令：

    ```bash
    /mecha-story-generator
    ```

    **魔法时刻：**
    Claude 会立刻回应：
    > "系统启动... 视觉诗人已就位，摄影指导已待命。请选择你要讲述的故事..."

### 接下来的流程

这就想点菜一样简单：

1. **AI 问**：“老板，今天想拍个什么片子？沙漠机甲还是地下城探险？” -> **你选**：“来个沙漠的。”
2. **AI 问**：“给您出了个7个镜头的分镜，您瞅瞅节奏对不对？” -> **你回**：“准了。”
3. **AI 说**：“好嘞！这就给您生成即梦和可灵的专用提示词。”

**最终你得到的**：
直接复制粘贴到视频生成软件的一段段精准描述，包含了你也想不到的灯光、构图和材质细节。

## 第五幕：案例展示——《最后的补给线》

**用户操作：**
选择故事 A（《最后的补给线》） -> 确认分镜

**Visual Poet 输出（节选）：**
> **画面描述**：
> 在金色时刻（Golden Hour）的沙漠废土中，巨大的**工业黄 (#FFD700)** 运输机甲残骸半埋在沙丘中。
> 它的表面布满了氧化痕迹，但在阴影处，**海军蓝 (#0066CC)** 的应急灯光依然在微弱闪烁，仿佛在呼吸。
> 一个渺小的拾荒者站在机甲巨大的机械臂下，这种**垂直构图**极大地强化了人类的脆弱感。

**Cinematographer 输出：**
> **运镜**：Crane Up + Slow Dolly Push-In（升降镜头接缓慢推镜头）
> **转场**：Match Cut（匹配剪辑）——从机甲的圆形关节切换到拾荒者的护目镜特写。

**生成的提示词（直接用于可灵 Kling）：**

它不回避未来可能的黑暗面，但它也不放弃希望。
